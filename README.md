# АвтовозЛоадер

Мобильное приложение для Android — расчёт погрузки автовоза **Scania P-series 4×4** с надстройкой **Uçsuoğlu**.

---

## Функции

- Добавление до 8 автомобилей вручную или из библиотеки
- Автоматическое распределение по двум ярусам (с учётом высоты и массы)
- Расчёт нагрузок на переднюю ось и заднюю тележку (правило рычага)
- Визуализация схемы размещения (вид сбоку, цветовая шкала масс)
- Сохранение и история рейсов
- Тёмная тема, интерфейс на русском языке
- Настройки параметров автовоза (редактируемые)

## Сборка APK через GitHub Actions

### Шаг 1 — загрузить проект на GitHub

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/ВАШ_ЛОГИН/autovoz-loader.git
git push -u origin main
```

### Шаг 2 — скачать APK

1. Открыть вкладку **Actions** в репозитории
2. Выбрать последний запуск **Build APK**
3. В разделе **Artifacts** скачать `AutovozLoader-debug-N`
4. Установить `.apk` на телефон (нужно разрешить установку из неизвестных источников)

> ℹ️ Сборка запускается автоматически при каждом `git push` в ветку `main`.

## Параметры по умолчанию (Scania P-series 4×4 + Uçsuoğlu)

| Параметр | Значение |
|---|---|
| Снаряжённая масса | 10 500 кг |
| Макс. полная масса | 44 000 кг |
| Нагрузка — передн. ось | 7 500 кг |
| Нагрузка — задн. тележка | 11 500 кг |
| Длина платформы | 12,0 м |
| Макс. высота — нижний ярус | 2,0 м |
| Макс. высота — верхний ярус | 1,8 м |

Все параметры редактируются в экране **Настройки**.

---

## Алгоритм расстановки

1. Автомобили сортируются: сначала те, что не помещаются на верхний ярус по высоте, затем по убыванию массы.
2. Заполнение нижнего яруса спереди назад, затем верхнего.
3. Нагрузки на оси считаются по **правилу рычага** с учётом положения центра тяжести груза.
4. Предупреждения при превышении допустимых нагрузок.
